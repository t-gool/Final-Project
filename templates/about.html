Over Mij

Welkom op de 'Over Mij'-pagina van mijn Transport Vergelijker! Hier vertel ik je meer over de applicatie en mijn visie achter het project.

Wat is de Transport Vergelijker?

Deze website heb ik gebouwd tijdens mijn minor creatief programmeren van de NHL Stenden. Dit is een externe minor die ik volg voor mijn afstudeerjaar van de opleiding Logistics Engineering. Zelf zit ik op de Breda University of Applied Sciences in Breda.

Waarom dit project?

Dit is mijn final project voor de minor creatief programmeren en wat ik heb gebouwd is een transporteursvergelijker. De website gaat een aantal vragen stellen aan je en vergelijkt de antwoorden van die vragen om zo de transporteur te kiezen die het beste daarop aansluit. Er zijn in de wereld tegenwoordig zoveel transporteurs dat het voor consumenten vaak lastig is welke nou de beste keuze is, terwijl er veel verschil is in prijzen en kwaliteit tussen deze transporteurs.

Technologieën en Tools

Deze applicatie is gebouwd met de volgende technologieën:

  • Flask - Een Python web framework voor het bouwen van de webapplicaties.
  • HTML/CSS - Voor de structuur en styling van de webpagina's.
  • JavaScript - Voor interactie, bijvoorbeeld knoppen
  • Python - Gebruikt voor de backend logica van de applicatie.
  • SQLite3 - Een simpele database voor het opslaan van gebruikersgegevens.
